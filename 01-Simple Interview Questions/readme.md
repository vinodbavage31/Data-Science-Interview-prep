## About This Folder

This folder contains **Python & Data Science interview questions** organized in two parts:

- **Part 1:** Questions 1 to 20  
- **Part 2:** Questions 21 to 40  

Each question includes a **Python solution with step-by-step explanations**, written in a **professional and interview-ready style** for easy practice and daily revision.

---

## Topics Covered

- **Basic Python Concepts:** variables, data types, loops, conditions, functions  
- **String Operations:** reversing strings, extracting letters, counting vowels, removing duplicates  
- **List, Tuple, and Dictionary Manipulations**  
- **Random Number Generation & Visualization**: `random`, `matplotlib` histograms  
- **Date & Time Operations**: using `datetime`  
- **Object-Oriented Programming**: classes, inheritance, multiple inheritance  
- **Conditional Logic & Problem Solving:** leap years, largest of three numbers, positive/negative checks  

---

## Usage

- Each question can be run in **Jupyter Notebook** or Python environment  
- Solutions are structured for **learning, revision, and interview practice**  
- You can explore Part 1 and Part 2 sequentially to cover all 40 questions
